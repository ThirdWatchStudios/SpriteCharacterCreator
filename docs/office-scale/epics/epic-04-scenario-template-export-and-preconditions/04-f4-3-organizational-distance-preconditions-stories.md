# F4.3 Organizational-Distance Preconditions - Stories

### S4.3.1 - Resolve The Distance Source Decision

User story:
- As both teams, I need the structural-vs-spatial distance decision resolved so the distance term reads a single agreed signal.

Acceptance criteria:
- The decision (org-chart structural distance vs. layout spatial distance, or both) is recorded with rationale.
- The chosen distance signal source is identified in Epic 1 (wing connectivity) or Epic 2 (org chart).
- The decision is shared with the sim.

Dependencies:
- F4.2.

### S4.3.2 - Add A Distance Precondition/Cost Term

User story:
- As a template author, I need a distance condition or distance-scaled cost so cross-wing difficulty and payload track organizational distance.

Acceptance criteria:
- A template can condition on, or be priced by, organizational distance.
- The term reads the agreed distance signal from S4.3.1.
- Casting honors the distance term.

Dependencies:
- S4.3.1.
